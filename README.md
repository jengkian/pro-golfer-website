# Pro Golfer Website

Premium golf club manufacturer website built with HTML, CSS, and vanilla JavaScript.

## Live Site

Visit: [https://jengkian.github.io/pro-golfer-website/](https://jengkian.github.io/pro-golfer-website/)

## Features

- Responsive single-page design (mobile, tablet, desktop)
- Product catalogue with 6 product lines
- Interactive product detail modals
- Scroll animations with Intersection Observer
- Contact form with client-side validation
- Accessible (ARIA labels, keyboard navigation, reduced-motion support)

## Tech Stack

- HTML5 / CSS3 / Vanilla JavaScript (ES6+)
- Google Fonts (Montserrat + Open Sans)
- CSS Custom Properties (design tokens)
- BEM naming convention
